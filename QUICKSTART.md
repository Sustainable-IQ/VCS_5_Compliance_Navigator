# Quick Start Guide

## Installation (First Time Only)

1. Open Command Prompt or PowerShell in the project folder
2. Run: `npm install`
3. Wait for installation to complete (2-3 minutes)

## Running the Application

1. Open Command Prompt or PowerShell in the project folder
2. Run: `npm run dev`
3. Browser will automatically open to http://localhost:3000

## Common Commands

```bash
# Install dependencies (first time only)
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Quick Tips

### Finding Specific Updates
- Use the search box at the top to search by ID, title, or keywords
- Filter by project type to see only relevant updates
- Filter by applicability (New Projects, Existing Projects, etc.)

### Understanding Effective Dates
- **Immediate**: Effective December 16, 2025
- **Jan 1, 2027**: Most common deadline for new projects
- **Jan 1, 2030**: Deadline for many existing project updates

### Using the Checklist Generator
1. Go to "My Checklist" tab
2. Fill in your project details
3. Click "Generate Checklist"
4. Export to PDF to share with your team
